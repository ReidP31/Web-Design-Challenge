Summary: Latitude vs. X

The purpose of this project was to analyze how different weather phenomena change as you approach the equator. Data was pulled from the OpenWeatherMap API for over 600 randomly selected cities and then converted to a Pandas DataFrame for ease of data visualization.

Data Visualization was performed by using Matplotlib, with the relationship between a city's latitude and its respective temperature, cloudiness, wind speed, and humidity plotted. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
